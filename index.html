Welcome to Playdroid Library Page.

This project aims to provide a set of advanced components and solutions to create your Android projects faster and easier.

Features:

  • Fully optimized ListView, GridView and Horizontal Gallery
    • Components which allow you to display a set of different cells, which will automatically manage all possible tasks such as image (downloading, caching, displaying), touch actions and view updates
  • ViewPager and Fragment ViewPager
  • Resource Manager
    • class which handles file management (download, storage, encryption)
  • Fragment Event Communication
    • Broadcasting event actions to groups of fragments to improve responsivity of your application.
  • Object oriented Database
  • Debug tools
    • CrashLogger, Internet speed monitor and automatic LogCat tagger.
  • Universal fragment and activity classes
    • Classes that contain all needed tools and methods which are used more than often :)
