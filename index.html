Welcome to Playdroid Library Page.

This project aims to provide a set of advanced components and solutions to create your Android projects faster and easier.

Features:

  • Fully optimized ListView, GridView and horizontal gallry, which allows you to display a set of different cells, which will automatically manage all possible tasks such as image (downloading, caching, displaying), touch actions and view updates
  • ViewPager and Fragment ViewPager
  • Resource Manager - class which handles file management (download, storage, encryption)
  • Fragment Event Communication - broadcasting event actions to groups of fragments to improve responsivity of your Application
  • Object oriented Database
  • Debug tools. Automatic CrashLogger, and LogCat utility
  • Universal fragment and activity classes which contain all needed tools and methods which are used more than often
